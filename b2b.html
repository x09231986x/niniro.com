FAQ

Вопросы о корпоративном
использовании

Как быстро можно подключить команду?
+
Создание корпоративного аккаунта и приглашение сотрудников занимает 5–10 минут. Никакой установки ПО — сотрудники сразу работают через браузер.
Можно ли ограничить бюджет для каждого сотрудника?
+
Да. В командном дашборде можно задать лимит расходов на пользователя или отдел. При приближении к лимиту — автоматическое уведомление администратору.
Как хранятся записи звонков? Это соответствует GDPR?
+
Записи и транскрипты хранятся на зашифрованных серверах в ЕС. Сервис соответствует требованиям GDPR. Enterprise-клиенты могут выбрать регион хранения данных. Все данные доступны только вашей организации.
Есть ли интеграция с CRM (Salesforce, HubSpot, Pipedrive)?
+
На тарифе Team доступны Webhook-уведомления и экспорт данных. Полный REST API и готовые интеграции с популярными CRM — на тарифе Enterprise. Список доступных интеграций уточняйте на демо.
Предоставляете закрывающие документы для бухгалтерии?
+
Да. Для юридических лиц предоставляем счета, акты и другие закрывающие документы. Работаем с компаниями из России, ЕС и других юрисдикций — уточните детали на демо.
Можно ли протестировать перед покупкой?
+
Да. Для команд мы проводим 30-минутное демо с живым тестовым звонком и настройкой под ваш сценарий. После демо — бесплатный тестовый период на 14 дней для всей команды.
